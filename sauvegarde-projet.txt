Date de création: 2025-01-24

================================================================================
DESCRIPTION DU PROJET
================================================================================
ZEEEMAX est une plateforme de branding et stratégie digitale qui accompagne 
les entrepreneurs à révéler leur identité de marque avec un branding 
sur-mesure et une stratégie digitale impactante.

Technologies utilisées:
- Laravel (Framework PHP)
- Blade (Moteur de templates)
- Tailwind CSS (Framework CSS)
- Alpine.js (Framework JavaScript)
- MySQL/MariaDB (Base de données)

================================================================================
STRUCTURE DE LA BASE DE DONNÉES
================================================================================

TABLES PRINCIPALES:
1. services - Services proposés par ZEEEMAX
2. portfolio_items - Réalisations et projets clients
3. testimonials - Témoignages clients
4. contact_messages - Messages de contact reçus
5. partners - Partenaires de ZEEEMAX
6. team - Membres de l'équipe
7. site_settings - Paramètres du site (un seul ou plusieurs enregistrements)
8. homepage_settings - Configuration de la page d'accueil
9. apropos - Contenu de la page À propos
10. blog_posts (newsletters) - Articles de blog / newsletters
11. valeurs - Valeurs de l'entreprise
12. subscribers - Abonnés à la newsletter
13. admins - Administrateurs du site
14. users - Utilisateurs du site
15. visits - Statistiques de visites

PROBLÈMES RÉSOLUS
================================================================================

1. PROBLÈME: Images ne s'affichaient pas (notamment logo-footer)
   CAUSE: Le fichier s'appelle logo-footer.PNG (majuscules) mais le code 
          référençait logo-footer.png (minuscules)
   SOLUTION: Modifié toutes les références dans le code pour utiliser 
             logo-footer.PNG (majuscules)
   FICHIERS MODIFIÉS:
   - resources/views/layouts/app.blade.php (3 occurrences)
   - resources/views/emails/newsletter.blade.php (1 occurrence)
   DATE: 2025-01-24

2. AJOUT: Données de test pour toutes les tables
   DESCRIPTION: Création d'un seeder complet (TestDataSeeder) qui génère 
                3 enregistrements pour chaque table de la base de données
   FICHIER CRÉÉ: database/seeders/TestDataSeeder.php
   FICHIER MODIFIÉ: database/seeders/DatabaseSeeder.php
   DATE: 2025-01-24

================================================================================
COMMANDES UTILES
================================================================================

Pour exécuter le seeder avec les données de test:
php artisan db:seed --class=TestDataSeeder

Pour réinitialiser la base de données et exécuter tous les seeders:
php artisan migrate:fresh --seed

Pour exécuter uniquement les migrations:
php artisan migrate

Pour voir les routes disponibles:
php artisan route:list
